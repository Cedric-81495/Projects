GrayScale Project Summary
Project Name

GrayScale

Project Type

Full-stack E-commerce Web Application

Technologies Used

Backend: Node.js, Express.js, MongoDB, Mongoose
Frontend: React, Vite, Tailwind CSS
State Management: Redux Toolkit
Authentication: JWT (JSON Web Tokens)
File Uploads: Cloudinary, Multer
Payment Methods: PayPal

Project Description

GrayScale is an online e-commerce platform for fashion products.
It allows managing products, users, orders, and newsletter subscriptions.

Admin Features:

Manage products (add, edit, delete)

View all users

View and manage orders

User Features:

Browse products and collections

Add products to cart

Checkout using or PayPal

View order history and order details

Frontend Features (React)

Home page with featured products and collections

Product listings and product detail pages

User registration, login, and profile management

Cart management and checkout process

Order confirmation and order history pages

Newsletter subscription form

Backend Features (Node.js + Express)

RESTful API endpoints for users, products, cart, orders, checkout, and subscriptions

Admin routes for product, order, and user management

JWT authentication and admin authorization

Image upload via Cloudinary

MongoDB database with Mongoose ODM

Input validation and error handling

Payment Methods

PayPal: Integrated for global payment support

Frontend Payment Components:
components/Cart/PayPalbutton.jsx

Notes

Backend handles all API operations and database interactions.

Frontend handles UI, user interactions, and communicates with the backend via API endpoints.

Middleware in backend protects private routes and checks admin permissions.

data/ folder is optional but useful for seeding initial data.

Assets in frontend should be organized logically (images, icons, logos) for maintainability.
